
# 🎓 Digital Learning Platform

## 📌 Overview

The Digital Learning Platform is a full-stack web application designed to provide an interactive learning experience through quizzes, performance tracking, and leaderboard competition. It allows users to test their knowledge, improve skills, and track progress in real time.

---

## 🚀 Features

* 👤 User Authentication (JWT-based login/signup)
* 📝 Multiple Quizzes with different categories
* ⏱️ Timer-based quiz system
* 📊 Leaderboard to track top performers
* 🎯 Difficulty levels (Easy, Medium, Hard)
* 🛠️ Admin Dashboard for managing quizzes
* 📂 PDF upload and learning resources

---

## 🛠️ Tech Stack

**Frontend:**

* HTML, CSS, JavaScript / React

**Backend:**

* Node.js, Express.js

**Database:**

* Firebase / MongoDB

## ▶️ Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/Deekshitha1035/Digital-Learning-Platform.git
```

### 2. Navigate to project folder

```bash
cd Digital-Learning-Platform
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the application

```bash
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the root and add:

```env
PORT=5000
JWT_SECRET=your_secret_key
FIREBASE_API_KEY=your_key
```
